# Variables and Outputs

The Terracina language includes a few kinds of blocks for requesting or
publishing named values.

- [Input Variables](/terracina/language/values/variables) serve as parameters for
  a Terracina module, so users can customize behavior without editing the source.

- [Output Values](/terracina/language/values/outputs) are like return values for a
  Terracina module.

- [Local Values](/terracina/language/values/locals) are a convenience feature for
  assigning a short name to an expression.
