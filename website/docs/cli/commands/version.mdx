# `terracina version` command

The `terracina version` command prints the current version of the Terracina binary and all
installed plugins.

## Usage

Usage: `terracina version [options]`

With no additional arguments, `version` displays the version of Terracina,
the platform it is installed on, installed providers, and the results of upgrade
and security checks unless disabled. Refer to [Upgrade and Security Bulletin Checks](/terracina/cli/commands#upgrade-and-security-bulletin-checks) for additional information.

## Flags

This command has one optional flag:

* `-json` - Formats version information as a JSON object. No upgrade or security information is included.

## Example

Basic usage, with upgrade and security information shown if relevant:

```shellsession
$ terracina version
Terracina v0.15.0
on darwin_amd64
+ provider registry.terracina.io/hashicorp/null v3.0.0

Your version of Terracina is out of date! The latest version
is X.Y.Z. You can update by downloading from https://www.terracina.io/downloads.html
```

As JSON:

```shellsession
$ terracina version -json
{
  "terracina_version": "0.15.0",
  "platform": "darwin_amd64",
  "provider_selections": {
    "registry.terracina.io/hashicorp/null": "3.0.0"
  },
  "terracina_outdated": true
}
```
