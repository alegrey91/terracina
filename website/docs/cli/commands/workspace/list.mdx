# `terracina workspace list` command

The `terracina workspace list` command lists all existing workspaces.

## Usage

Usage: `terracina workspace list [DIR]`

The command will list all existing workspaces. The current workspace is
indicated using an asterisk (`*`) marker.

## Example

```
$ terracina workspace list
  default
* development
  jsmith-test
```
