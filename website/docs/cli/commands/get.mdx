# `terracina get` command

Run the `terracina get` command to download and update
[modules](/terracina/language/modules/develop) declared in the root module.

## Usage

Usage: `terracina get [options]`

The modules are downloaded into a `.terracina` subdirectory of the current
working directory. Don't commit this directory to your version control
repository.

The `get` command supports the following option:

* `-update` - If specified, modules that are already downloaded will be
  checked for updates and the updates will be downloaded if present.

* `-no-color` - Disable text coloring in the output.
